BMI Calculator: Calculates Body Mass Index and provides an instant health category status (Underweight, Normal, Overweight, or Obese).

Random Array Picker: Generates a random array and extracts the first and last elements using JavaScript.

Live Addition (NaN Handling): Real-time math using event listeners. Displays NaN until both inputs are valid, demonstrating JavaScript's type-handling.
